# docs/archive/

이 디렉토리는 rack-labs/rack-tracker에서 마이그레이션된 문서 아카이브다.

## 아카이브 출처

| 항목 | 내용 |
|------|------|
| 출처 레포 | https://github.com/rack-labs/rack-tracker |
| 관계 | 마이그레이션 출처. rack-venture/rack-tracker의 git upstream이 아니다. |
| 운영 주체 | rack-labs (졸업프로젝트 팀), rack-venture와 별도 운영 |

## 주의

- 아카이브 내 이슈 번호는 rack-labs/rack-tracker 기준이다. rack-venture 이슈 번호와 다르다.
- 이 디렉토리 아래 문서는 수정하지 않는다.
- rack-venture 신규 작업 문서는 `docs/mvp-v2/` 또는 `docs/issues/` 아래에 만든다.
